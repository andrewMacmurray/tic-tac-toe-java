### To play the games:

Build both projects using:

```sh
> ./gradlew assemble
```

You can then run the Command Line version using:

```$xslt
> java -jar build/libs/CliTicTacToe-1.0-SNAPSHOT.jar
```

Or the JavaFx version using:

```$xslt
> java -jar build/libs/FxTicTacToe-1.0-SNAPSHOT.jar
```
